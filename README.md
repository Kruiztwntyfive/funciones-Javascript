# Funciones Javascript - Grupo A

Este repositorio contiene ejercicios y ejemplos prácticos desarrollados como parte del "CURSO FULLSTACK / TALLER HTML Y CSS". El objetivo principal es explorar conceptos fundamentales de JavaScript y TypeScript, incluyendo el manejo de variables, funciones y tipado.

## Estructura del Proyecto

El proyecto está organizado en las siguientes carpetas:

- **assets/**: Contiene recursos estáticos compartidos.
- **ts/**: Código fuente y ejemplos escritos en TypeScript.
- **variables/**: Ejemplos básicos sobre la declaración y uso de variables.
- **variables-constantes/**: Ejercicios específicos sobre tipado dinámico, constantes y manipulación del DOM (incluye archivos HTML, CSS y JS).

## Requisitos

- Node.js
- TypeScript (configurado como dependencia de desarrollo)

## Instalación

1. Clona este repositorio o descarga los archivos.
2. Abre una terminal en la raíz del proyecto.
3. Instala las dependencias necesarias ejecutando:

```bash
npm install
```

## Cómo usar

Puedes explorar los ejemplos navegando por las carpetas. Por ejemplo, para ver el ejercicio de tipado dinámico:

1. Ve a la carpeta `variables-constantes/`.
2. Abre el archivo `tipado-dinamico.html` en tu navegador web preferido.

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
- TypeScript
