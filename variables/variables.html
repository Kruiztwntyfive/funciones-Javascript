Creación y Comandos


Declaración Asignación Inicialización

Es el ciclo de vida de una variable. La Declaración crea el nombre. La Asignación (=) da o cambia el valor. La Inicialización Se refiere específicamente a la creación de un nuevo objeto a partir de una plantilla predefinida (una Clase), típicamente usando la palabra clave new.

let, const, var

let, const y var palabras clave de JavaScript para declarar variables, pero difieren en su alcance y mutabilidad. var es la forma antigua con alcance de función y reasignación; let tiene alcance de bloque (se limita a llaves {}) y permite reasignar valores; const también tiene alcance de bloque, pero su valor no se puede reasignar una vez asignado.

Tipos Literales


Tipo Number (Integer & Float)

Único tipo para números enteros (Integer) y decimales (Float). Se usa para cálculos matemáticos.

Tipo String

Cadena de caracteres, usada para manipular texto. Se define con comillas simples ('') o dobles ("").

Tipo Boolean

Solo tiene dos valores posibles: true o false. Fundamental para la lógica condicional del código.

Null y Undefined

Ambos indican la ausencia de valor. Null es intencional (asignado por el programador). Undefined es el valor por defecto (aún no se asigna valor).
